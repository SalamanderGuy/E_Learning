Η γλώσσα C σε βάθος Τελικό Διαγώνισμα
Επανάληψη των προηγουμένων
1. Τι πρέπει να επιστρέφει ένα πρόγραμμα κατά την επιτυχή εκτέλεση του, στο λειτουργικό σύστημα;



2. Ποιά είναι η μόνη συνάρτηση που πρέπει να έχουν όλα τα προγράμματα C;



3. Ποιό ζεύγος συμβόλων ορίζει την αρχή και το τέλος ενός block κώδικα;



4. Με ποιό σύμβολο τελειώνουν οι περισσότερες γραμμές C κώδικα;



5. Ποιό από τα παρακάτω είναι ένα σωστό σχόλιο;



6. Ποιό από τα παρακάτω δεν είναι σωστός τύπος δεδομένων;



7. Ποιός από τους παρακάτω είναι ο σωστός τελεστής για την σύγκριση δύο μεταβλητών;



8. Ποιός από τους παρακάτω είναι ο σωστός τρόπος για να ορίσουμε μια μεταβλητή;



9. Τι θα τυπώσει το παρακάτω πρόγραμμα;
int main() {
int a = 5;
int b = 6;
int c = a + b;
printf("%d", c);
}



10. Τι θα τυπώσει το παρακάτω πρόγραμμα;
int main() {
int a = 2;
int b = 33;
if (a <= b) {
printf("%d", 42);
}
else printf("%d", 3);
}
